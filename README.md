📊 Stack Overflow Developer Survey: Market Insights Analysis 2026
Про проєкт
Цей репозиторій містить комплексний аналіз глобального ринку IT на основі щорічного опитування Stack Overflow. Проєкт охоплює шлях від базової описової статистики до побудови стратегічних теплових карт доходів розробників.

Це навчальний проєкт, виконаний у межах курсу з аналітики даних, де технічні розрахунки поєднуються з бізнес-інтерпретацією результатів.

🛠 Ключові етапи та технічні рішення
Проєкт розбитий на 10 аналітичних завдань, що охоплюють:

Обсяг даних: Підрахунок загальної кількості респондентів для розуміння масштабу дослідження.

Валідація повноти: Аналіз "ідеальних респондентів", які відповіли на всі критичні питання (через інтерсекцію множин).

Досвід роботи: Розрахунок мір центральної тенденції (Mean, Median, Mode) для поля WorkExp.

Remote Work: Статистика поширеності віддаленої роботи.

Екосистема Python: Розрахунок частки Python-розробників на глобальному ринку.

EdTech аналіз: Дослідження впливу онлайн-курсів на підготовку кадрів.

Географія доходів: Аналіз середньої та медіанної компенсації Python-спеціалістів у розрізі країн.

Education vs Salary: Дослідження рівнів освіти 5 найбільш оплачуваних спеціалістів світу.

Вікова популярність: (Власний аналіз) Розрахунок популярності Python для різних вікових категорій з корекцією методології знаменника.

High-Value Industries: Визначення топових індустрій для спеціалістів 75-го перцентиля, що працюють віддалено.

**Мета:** Візуалізація медіанних зарплат для понад 30 мов та 15 індустрій.
**Методологія:** Використано фільтрацію за квантилями (**0.25–0.75**) для усунення викидів та метод `.explode()` для коректного обліку кожної мови в мульти-стек респондентів.

* **Зелена зона (Прибуткові):** Комбінації [Мова] + [Індустрія], що виділені зеленим, демонструють найвищу віддачу на інвестиції в навчання. Це найкращий вибір для старту або зміни кар'єри в 2026 році.
* **Червона зона (Слабкі):** Вказує на індустрії з обмеженим бюджетом або застарілим стеком. Аналітика показує, що перехід у ці ніші наразі менш вигідний.
